# 板块数据下载与选择指南

本文档介绍如何按板块下载股票数据，并在前端界面中选择板块进行回测。

## 功能概述

- **按板块下载**：支持 8 个主要行业板块，批量下载股票历史数据
- **文件标识**：下载的 CSV 文件自动添加板块前缀，便于识别和管理
- **前端选择**：Web 界面支持可视化选择板块，无需手动输入股票代码

## 一、下载板块数据

### 1.1 下载单个板块

```bash
npm run download:sector
```

运行后按提示选择：
1. 选择数据源（推荐 `netease`）
2. 选择要下载的板块（支持多选）
3. 输入起始日期（默认 20200101）
4. 输入结束日期（默认 20991231）

### 1.2 下载所有板块

```bash
npm run download:all
```

一次性下载全部 8 个板块的股票数据。

### 1.3 查看板块列表

```bash
npm run download:list
```

显示所有可用板块及其包含的股票代码。

## 二、数据文件结构

下载的数据保存在 `data/` 目录下，文件命名格式：

```
{板块代码}_{股票代码}.csv
```

示例：
```
data/
├── bank_600000.csv      # 银行板块 - 工商银行
├── bank_600036.csv      # 银行板块 - 招商银行
├── tech_000063.csv      # 科技板块 - 中兴通讯
└── consumer_600519.csv  # 消费板块 - 贵州茅台
```

### CSV 文件格式

```csv
date,open,high,low,close,volume
20240102,10.50,10.80,10.45,10.75,1000000
20240103,10.75,11.00,10.70,10.95,1200000
...
```

### 元数据文件

每个 CSV 文件会附带一个同名的 `.meta.json` 文件，记录下载信息：

```json
{
  "symbol": "600000",
  "sector": "bank",
  "filename": "bank_600000.csv",
  "downloadDate": "2026-04-12T10:30:00.000Z",
  "source": "netease",
  "barCount": 1000
}
```

## 三、前端使用板块选择

### 3.1 启动服务

确保后端和前端服务都在运行：

```bash
# 后端（项目根目录）
npm run server

# 前端（web 目录）
cd web && npm run dev
```

访问 http://localhost:5173

### 3.2 选择板块数据

1. **选择数据源类型**
   - 在左侧边栏找到"数据源"面板
   - 选择数据源类型为 `CSV 目录`

2. **点击板块选择**
   - 点击"板块选择"按钮，弹出板块选择对话框

3. **勾选板块**
   - 勾选需要的板块（支持多选）
   - 每个板块显示包含的股票数量
   - 底部显示选中的股票总数

4. **确认选择**
   - 点击"确认选择"按钮
   - 系统自动加载所选板块的所有股票数据

5. **运行回测**
   - 选择策略并配置参数
   - 点击"运行回测"开始分析

## 四、自定义板块配置

如需添加或修改板块配置，编辑 `sectors/index.ts`：

```typescript
export const SECTORS: SectorConfig[] = [
  {
    name: '新能源',        // 板块名称
    code: 'newenergy',     // 板块代码（唯一）
    symbols: [             // 股票代码列表
      '002594', '300750',  // ...
    ],
  },
  // ... 更多板块
];
```

修改后同步更新前端的 `web/src/types/backtest.ts` 中的 `SECTOR_OPTIONS`。

## 五、常见问题

### Q: 下载失败或数据不完整？
A: 检查网络连接，更换数据源（`eastmoney` 或 `sina`）重试。

### Q: 前端看不到某些股票？
A: 确保 CSV 文件已下载到正确的 `data/` 目录，且文件名包含板块前缀。

### Q: 如何清除已下载的板块数据？
A: 手动删除 `data/` 目录下的相关 CSV 文件即可。

### Q: 板块配置在哪里修改？
A: 后端配置在 `sectors/index.ts`，前端配置在 `web/src/types/backtest.ts`。
