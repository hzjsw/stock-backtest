# 🎉 QuantX 股票量化回测系统 - 功能总览

## ✨ 已完成功能

### 📊 1. 完整的回测引擎

**核心功能**:
- ✅ 市价单/限价单撮合
- ✅ 滑点模拟
- ✅ 手续费 + 印花税计算
- ✅ 仓位管理
- ✅ 防止未来数据泄漏
- ✅ 完整的绩效指标计算

**支持策略**:
| 策略 | 参数 |
|------|------|
| 双均线交叉 | 短期周期、长期周期 |
| MACD | 快线、慢线、信号线 |
| RSI | 周期、超买/超卖阈值 |
| 布林带 | 周期、标准差倍数 |
| 多因子选股 | 4 个因子权重、持仓数量 |

**技术指标库**:
- SMA, EMA, MACD, RSI, 布林带，ATR, KDJ, VWAP

**因子选股**:
- 动量因子、波动率因子、换手率因子、市值因子、PE/PB因子、均线偏离因子、量比因子
- 多因子打分系统

### 🌐 2. 数据源管理（4 种方式）

#### 方式 1: 模拟数据
- 随机生成 500 日 K 线
- 适合快速测试、演示

#### 方式 2: CSV 单文件
- 拖拽上传 CSV 文件
- 自动验证格式
- 适合单只股票回测

#### 方式 3: CSV 目录
- 批量加载目录下所有 CSV
- 适合多因子选股策略
- 股票池回测

#### 方式 4: 在线获取 ⭐ 新功能
- **东方财富**: 免费、历史数据完整、无需 token
- **新浪财经**: 实时数据、速度快
- **Tushare**: 数据质量高、需 token

**使用示例**:
```typescript
import { fetchMultipleStocks } from './src/data/web-loader';

const data = await fetchMultipleStocks(
  ['600000', '600036', '000001'],
  'eastmoney',
  { startDate: '20230101', endDate: '20231231' }
);
```

## 🚀 快速开始

### 一键启动
```bash
# Windows
start-web.bat

# 手动启动
npm run server    # 后端
cd web && npm dev # 前端
```

访问 http://localhost:5173

### 使用在线数据回测

1. 访问 Web UI
2. 左侧 → 数据来源 → **在线获取**
3. 选择数据源：**东方财富**
4. 输入股票代码：`600000, 600036, 000001`
5. 设置日期：`20230101` - `20231231`
6. 点击"**获取数据**"
7. 选择策略：双均线交叉
8. 点击"**运行回测**"
