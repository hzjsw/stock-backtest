# v2.0 - 策略优化和代码质量改进

## 🎯 新增功能

### 1. 策略优化工具
- **网格搜索** - 自动搜索最优参数组合
- **策略对比** - 同时对比 7 种策略绩效
- **排名系统** - 按收益率自动排序

```bash
npm run example:optimize
```

### 2. 单元测试框架
- 35 个测试用例覆盖核心模块
- 技术指标、回测引擎、数据加载、日志系统

```bash
npm run test:run
```

### 3. 日志和错误处理
- 结构化日志输出
- 自定义错误类型
- API 增强的错误响应

### 4. 参数范围扩展
- MA 长期均线最高 300 日
- RSI 超卖阈值最低 5
- MACD 慢线周期最高 150

## 📦 安装和使用

```bash
# 克隆仓库
git clone https://github.com/hzjsw/stock-backtest.git
cd stock-backtest

# 安装依赖
npm install

# 运行策略优化
npm run example:optimize

# 启动 Web 界面
npm run server    # 后端
cd web && npm run dev  # 前端

# 运行测试
npm run test:run
```
