# 数据源配置指南

## 🎯 三种数据源模式

### 1. 模拟数据 (Mock Data)

**适用场景**:
- 快速测试策略逻辑
- 演示系统功能
- 没有真实数据时

**特点**:
- ✅ 无需准备数据
- ✅ 自动生成 500 日 K 线
- ❌ 数据随机，不可用于实盘参考

**使用方法**:
```
Sidebar → 数据来源 → 选择"模拟数据"
```

---

### 2. 单 CSV 文件模式

**适用场景**:
- 回测单只股票
- 测试特定股票策略

**特点**:
- ✅ 使用真实历史数据
- ✅ 结果可靠可验证
- ❌ 每次只能回测一只股票

**使用方法**:

1. 准备 CSV 文件（如 `600000.csv`）
2. 在 UI 中选择"单文件"模式
3. 拖拽文件到上传区域或点击选择
4. 系统自动识别股票代码（文件名）

**CSV 格式**:
```csv
date,open,high,low,close,volume
2023-01-03,13.08,13.22,13.01,13.18,98560000
2023-01-04,13.20,13.35,13.15,13.30,102340000
```

---

### 3. CSV 目录模式 (推荐)

**适用场景**:
- 多因子选股策略
- 股票池回测
- 批量测试

**特点**:
- ✅ 支持多只股票
- ✅ 适合多因子策略
- ✅ 一次回测整个股票池

**使用方法**:

1. 创建目录（如 `data/`）
2. 放入多个 CSV 文件：
   ```
   data/
   ├── 600000.csv
   ├── 600036.csv
   ├── 000001.csv
   └── 000002.csv
   ```
3. 在 UI 中选择"CSV 目录"模式
4. 输入目录路径：`data/` 或 `examples/`

**支持的策略**:
- ✅ 多因子选股策略（需要多只股票）
- ✅ 单只股票策略（自动选择第一只）

---

## 📊 数据验证

系统会自动验证 CSV 文件：

| 检查项 | 说明 |
|--------|------|
| ✅ 文件存在 | 路径正确且可访问 |
| ✅ 必需列 | 包含 date, open, high, low, close, volume |
| ✅ 数据行数 | 至少有一行有效数据 |
| ✅ 日期格式 | YYYY-MM-DD |
| ✅ 数值解析 | 价格/成交量可转为数字 |

**验证失败会显示错误提示**。

## 🛠️ 获取真实数据

### Tushare (推荐)

```python
import tushare as ts

pro = ts.pro_api('your_token')
df = pro.daily(ts_code='600000.SH', start_date='20230101', end_date='20231231')
df = df[['trade_date', 'open', 'high', 'low', 'close', 'vol']]
df = df.rename(columns={'trade_date': 'date', 'vol': 'volume'})
df.to_csv('600000.csv', index=False)
```

### AKShare (免费)

```python
import akshare as ak

df = ak.stock_zh_a_hist(symbol="600000", period="daily")
df = df[['日期', '开盘', '最高', '最低', '收盘', '成交量']]
df.columns = ['date', 'open', 'high', 'low', 'close', 'volume']
df.to_csv('600000.csv', index=False)
```

## ⚠️ 注意事项

### 数据质量

1. **数据量**: 建议至少 60 个交易日（3 个月）
   - 技术指标需要足够历史数据
   - 少于 30 日可能无法计算某些指标

2. **停牌处理**: 
   - 停牌日可以跳过
   - 或使用前一日收盘价

3. **复权**: 
   - 建议使用复权数据（前复权/后复权）
   - 保证价格连续性

4. **价格异常**:
   - 检查是否有价格为 0 或负数
   - 检查是否有极端异常值

### 文件路径

- **相对路径**: 相对于 `stock-backtest/` 目录
- **绝对路径**: 完整路径（推荐 Windows 使用 `D:/data/` 格式）

### 多因子选股策略

使用多因子策略时**必须**提供多只股票数据：
- 最少 3 只股票
- 推荐 10 只以上
- 使用 CSV 目录模式最方便
